\begin{description}
\item status \\ 

\item stagingしたあとのdiff
\begin{verbatim}
   git diff --cached
\end{verbatim}

\item unstaging
\begin{verbatim}
   git checkout -- <filename>
\end{verbatim}
変更を残しておきたい場合はstashを用いる．
\\

\item commit \\
   git commit $\<$-a$\>$
   \\
  \textcolor{red}{
     -aを付けるとstagingしていないファイルも
     commitの対象となるので要注意！
  }
